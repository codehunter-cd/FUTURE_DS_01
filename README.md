# 📊 Sales Performance Dashboard – Power BI Project

## 🛠 Tools & Technologies Used
- Power BI – Interactive dashboard creation
- Power Query – Data transformation & cleaning
- Excel / CSV – Data source handling
- Data Visualization – KPI tracking and trend analysis

---

## 📈 Key Insights
- 💻 Technology category generated the highest revenue
- 📅 Sales showed significant growth during peak seasonal periods
- 🌍 West region achieved the best overall performance
- ⚠️ Certain products recorded negative profit margins
- 📦 High sales volume does not always guarantee profitability

---

## 📌 Recommendations
- ✅ Focus marketing and inventory efforts on high-performing categories
- 📍 Improve sales strategies in underperforming regions
- 💰 Optimize pricing and discount policies to improve profit margins
- 📊 Monitor loss-making products and reduce operational inefficiencies
- 🚀 Use customer and regional insights for targeted business decisions

## 🎯 Project Objective
This dashboard analyzes sales performance across regions, product categories, and customer segments to help businesses make data-driven decisions and improve profitability.

## 📂 Dashboard Features
- Interactive filters and slicers
- Region-wise sales analysis
- Category & sub-category performance tracking
- Profit and revenue comparison
- Seasonal sales trend visualization
- KPI cards for quick insights
